Under Development

The basic idea under development is to create an online "cache" of navdata and the like

The project is currently at http://github.com/ac001/fg-navdata/tree/master

The Goal

  • An online service where data can be constantly pulled and is updated
  • End users wont have to parse out files or create database and import etc, it can be pulled from the Google cloud
  • Data available in different formats, eg json, xml, yaml, csv
